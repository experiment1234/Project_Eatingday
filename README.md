파일 폴더들이 생길꺼에요

자동으로 git 로컬 저장소가 생겨요

열심히 개발을 하고  add, commit, push를 해요 

push 시 주의 사항은 dev [브랜치]에 해요 

git checkout dev 하면 (main) 이라고 되어있던 부분이 (dev) 라고 바뀌어요.
(안되면 git remote update
        git checkout 브랜치명)
git push origin dev 하면 됩니다.
